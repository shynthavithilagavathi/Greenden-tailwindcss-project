🌱 Greenden Project

Greenden is a modern and elegant web project built using **HTML** and **Tailwind CSS**.  
It showcases a clean, responsive design with utility-first styling, perfect for practicing front-end development with Tailwind.


📌 Features
 Responsive layout powered by Tailwind’s utility classes
 Nature-inspired design theme
 Semantic HTML structure
 Easy to extend and customize with Tailwind
 

 🛠 Technologies Used
 HTML5 – for structure
 Tailwind CSS – for styling and layout
